
# 🧠 核心心法：记牌与算牌（基本功）
进阶玩家和新手最大的区别在于记牌。你不需要像机器人一样记住每一张牌，但必须掌握关键信息：
记大牌（10以上）：
    - 必须记住王、级牌（逢人配）、A、K出了多少张。
    - 目的： 判断外面是否还有炸弹，以及谁手里可能握着“同花顺”或“大炸弹”。
记断张：
    - 如果你手里没有某张牌（比如5），且桌面上也没人打过，那么外面大概率有5的炸弹或同花顺。
记关键张（7和10）：
    - 7是顺子的核心（3-7, 4-8, 5-9, 6-10, 7-J）。如果7出完了，顺子就难组了。
    - 10是连接中张和大牌的桥梁。
算剩余张数：
    - 当有人出牌时，如果只剩几张，要根据之前的出牌习惯推测他的牌型。比如他剩3张，大概率是“三带二”或者“单张+对子”，或者是“小炸弹”。

# 🤝 配合艺术：上家与下家（战略）
掼蛋是2v2的游戏，配合大于个人技术。
如果你是“上家”（坐在对手的下手）
任务： 顶牌（卡位）。
策略： 你的首要任务不是自己走，而是不让下家（你的队友）难受。
    - 如果地主/对手出小牌，你要尽量出大一点的牌把路堵死，逼对手拆牌或炸。
    - 千万不要出队友管不上的牌（比如你出个极小的单张，队友手里全是大的，这就很尴尬）。
    - 口诀： 上家顶大牌，下家过小牌。
如果你是“下家”（坐在对手的上手）
任务： 接风与送牌。
策略： 你要时刻准备接队友的牌，或者给队友喂牌。
    - 如果队友出了单张，你手里有单A，不要轻易拆对子去管，除非为了拿回出牌权。
    - 一旦队友出完牌，你要争取拿到“接风”的机会（即最后一手牌没人要），这样下一轮你先出，可以打出队友需要的牌型。
信号传递
出牌暗示：
    - 如果你手里对子多，尽量先打对子，暗示队友留对子。
    - 如果你突然打出一张很大的单牌（如K或A），通常意味着你单牌很强，或者想让队友过牌。

# 🃏 牌型运用技巧（战术）
炸弹的使用
不要为了炸而炸： 炸弹是战略资源。除非能拿回出牌权并确立优势，或者为了阻止对手冲刺，否则不要轻易扔炸弹。
早炸与晚炸：
    - 早炸： 手里牌很烂，或者为了破坏对手的牌型结构（如拆掉对手的同花顺）。
    - 晚炸： 手里牌很整齐，想留到最后冲刺或拿双下。
逢人配（红桃级牌）的运用：
    - 百搭牌不要乱用： 不要轻易把“配牌”用在凑小顺子或三带上。
    - 最强用法： 凑同花顺或四大头炸。
    - 次强用法： 凑六头炸或五头炸来控场。
牌型的取舍
拆牌的艺术：
    - 如果手里有AAAA+K，对手出单张，你可以拆A去顶，保留KKK或KK。
    - 如果手里顺子太多，导致单张太散，不如拆掉顺子凑成“三带二”或“钢板”，因为顺子容易被大顺子管死，而三带二比较灵活。

# 🎯 局势判断（大局观）
头游与二游
争头游： 如果队友牌好，你要全力配合他争头游，哪怕牺牲自己的牌。
保二游： 如果对手一家牌特别好（要冲头游），你和队友要尽量保住二游，避免被“双下”（最惨的结果）。
    - 策略： 这种情况下，哪怕你有炸弹，如果炸了也拿不到头游，不如留着炸对手的最后一手，帮队友挡一下，让队友走二游。
残局处理
当有人剩牌很少（1-3张）时，必须高度警惕。
剩1张： 肯定是单张，用你最小的能管上的牌去管，或者直接用大牌封死。
剩2张： 可能是对子，也可能是两张单。
剩3张： 可能是三带二（手里剩三张一样的），也可能是单+对。

# 📌 进阶口诀总结
火不轻发： 炸弹不到关键时刻不打。
上家顶，下家顺： 位置决定打法。
配牌不凑小： 逢人配尽量凑大牌或炸弹。
宁丢头游，不丢双下： 团队利益最大化。
七死八活： 7和8是顺子的命门，盯紧这两张牌。
最后建议： 多复盘。打完一把，回想一下哪张牌出错了，或者哪个炸弹应该早一点扔。掼蛋打的是概率，也是心理战。
